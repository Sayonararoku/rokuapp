About the Application

Nerdplayer App is a media player designed to provide a fast, simple, and reliable experience on Roku devices.

The application allows users to access and organize content from compatible external sources using their own credentials and configurations.

Nerdplayer App does not provide, host, or distribute audiovisual content. All content played within the application is added and managed exclusively by the user.
